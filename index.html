Github Watcher by jmhobbs

Sync and watch your github watch list.

Github Watcher is a GTK+ desktop application that monitors your Github watch list and keeps a synced copy of each watched repository on your local machine. Not really. It does everything it is supposed to, up to a point. I haven't got it all the way debugged, and to be honest it needs some re-writes because I was in a hurry the first time. But it will download and sync your repos, just not consistently over a long period of time.

Dependencies

There may be others I'm not remembering, but here are some definites. * Python 2.5-ish * py-github * Git Python * PyGTK 2.0 * A Unix type system

Install

Clone the repo and run watcher.py
